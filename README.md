# News App
- This app downloads and shows the news to the user

# Features
- Unit and integration tests using JUnit, dagger hilt and Robolectric
- Jetpack compose UI tests

# Demo
https://github.com/user-attachments/assets/cec1d258-9c79-4edf-8bae-0605df3f95b7

# Installation
Build the app on Android Studio and launch on a connected device
(adb debugger). Or install the APK directly on the device.

# Usage
- Open the app and read the news

# Tech Stack
- Language: Kotlin
- UI: Jetpack Compose

# Environment
- Android Studio Koala Feature Drop | 2024.1.2 Patch 1
- Compile/Target sdk = 33
- min sdk = 24
- Java version 23
